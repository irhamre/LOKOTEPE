Repositori ini berisi kumpulan kode sumber dan dokumentasi untuk pengembangan sistem pemantauan tekanan ban (TPMS) yang menggunakan kombinasi modul ESP32, LilyGO T3, dan Nordic nRF52840 DK.Struktur Repositori1. Gateway & Node (Sistem Komunikasi P2P)Gate_Test_P2P: Program untuk LilyGO T3 yang berfungsi sebagai Gateway. Mengimplementasikan fitur radio diagnostic untuk memantau data dari 12 ban.Node_Test_P2P: Program untuk ESP32 + BLE + CC1101 yang berfungsi sebagai Node. Bertugas mengambil data TPMS dari 12 ban dan mengirimkannya melalui protokol P2P.gate v2: Program Gateway versi terbaru (v2) pada LilyGO T3, yang telah dioptimalkan untuk menerima dan memproses data dari 30 sensor TPMS dengan fitur radio diagnostic.node v2: Program Node versi terbaru (v2) pada ESP32 + CC1101, yang dirancang untuk mengambil data dari 30 sensor TPMS dengan fitur radio diagnostic.2. Modul Pembacaan SensorTPMS_Nordic: Program khusus untuk membaca data dari 12 sensor TPMS menggunakan platform Nordic nRF52840.3. DokumentasiTPMS nRF on Arduino IDE.pdf: Panduan langkah demi langkah (tutorial) mengenai cara memprogram board Nordic nRF52840 DK menggunakan Arduino IDE, termasuk konfigurasi board support package, bootloader, dan pemecahan masalah (troubleshooting) port.Ringkasan PerangkatKomponenPeranSpesifikasi UtamaLilyGO T3GatewayRadio Diagnostic, P2P CommunicationESP32 + CC1101NodeBLE, Sensor Data Acquisition, P2PnRF52840 DKSensor Reader12 Sensor TPMS ProcessingMemulai (Getting Started)Untuk Pengembangan Nordic nRF52840:Silakan merujuk pada file TPMS nRF on Arduino IDE.pdf di repositori ini untuk instruksi setup lingkungan Arduino IDE yang benar.Untuk Pengembangan ESP32/LilyGO:Pastikan library CC1101 dan library pendukung komunikasi radio telah terinstal di Arduino IDE Anda.Sesuaikan konfigurasi pin pada file kode (jika menggunakan hardware custom).
